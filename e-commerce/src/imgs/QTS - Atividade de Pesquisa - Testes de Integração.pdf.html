QUALIDADE E TESTE DE SOFTWARE ATIVIDADE DE PESQUISA - TESTES DE INTEGRAÇÃO Individualmente, realize uma pesquisa sobre testes de integração no desenvolvimento de software. A pesquisa deve incluir: 1. Conceito de Testes de Integração: Explique brevemente o que são testes de integração e qual sua importância no desenvolvimento de sistemas. 2. Objetivo dos Testes de Integração: Comente como esses testes ajudam a garantir a interação correta entre diferentes módulos de um sistema. 3. Explique o que é a técnica de mocking e como ela pode ser usada nos testes de integração no Javascript com o JEST. Entrega: A pesquisa deve ser entregue em formato (PDF), contendo as respostas de cada tópico e uma conclusão de 3 a 4 linhas sobre a importância dos testes de integração para a qualidade de software.
